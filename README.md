# EcoGit 🍃 - Manual GitHub para o Agro Sustentável

## 🎯 Objetivo do Projeto
Este projeto foi desenvolvido como submissão para a categoria de Programação Front-End do **Concurso Agrinho**, sob o tema **“Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente”**.

O site atua como uma ferramenta de inclusão digital e educacional, fornecendo um guia passo a passo, acessível e responsivo sobre como utilizar o **GitHub**. O foco principal é capacitar estudantes e produtores rurais a utilizarem a maior plataforma de código aberto do mundo para armazenar, versionar e compartilhar soluções tecnológicas voltadas à sustentabilidade no campo (como códigos de sensores IoT para manejo hídrico eficiente e IA aplicada à lavoura).

## 🚀 Funcionalidades Principais (Critérios de UX/UI)
- **Manipulação Ativa do DOM:** Saudação personalizada ao usuário em tempo real a partir de entrada de texto.
- **Acessibilidade Dinâmica:** Botões dedicados ao controle de tamanho de fontes (`A+` e `A-`) via JavaScript para usuários com baixa visão.
- **Alternância de Tema (Modo Escuro):** Respeito à preferência visual e economia de energia de dispositivos em ambientes de campo.
- **Responsividade Total:** Interface adaptada fluidamente para telas desktop, tablets e smartphones usando CSS Grid, Flexbox e Media Queries.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estruturação semântica de formulários, cards interativos e tags de acessibilidade.
- **CSS3:** Estilização com propriedades de layout modernas, variáveis de cores sustentáveis e animações suaves de transição no estado `:hover` dos botões.
- **JavaScript (ES6):** Lógica pura para manipulação de eventos e estados do usuário sem bibliotecas externas.
